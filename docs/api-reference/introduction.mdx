# API Reference

Welcome to the WebLinq API documentation. Our REST API enables powerful web automation, content extraction, and data transformation capabilities.

## Base URL

```bash
https://api.weblinq.dev/v1
```

## Authentication

All endpoints require authentication using a Bearer token:

```bash
curl -H "Authorization: Bearer YOUR_API_KEY" https://api.weblinq.dev/v1/...
```

<Note>Get your API key from the [dashboard](https://weblinq.dev/dashboard/api-keys).</Note>

## Quick Start

<Steps>
  <Step title="Get your API key">
    [Create an API key](https://weblinq.dev/dashboard/api-keys) from your dashboard.
  </Step>
  <Step title="Make your first request">
    Try extracting content from a webpage:
    
    ```bash
    curl -X POST "https://api.weblinq.dev/v1/web/markdown" \
      -H "Authorization: Bearer YOUR_API_KEY" \
      -H "Content-Type: application/json" \
      -d '{"url": "https://example.com"}'
    ```
  </Step>
  <Step title="Explore endpoints">
    Check out the [overview](/api-reference/overview) for all available endpoints.
  </Step>
</Steps>

## Response Format

All responses follow a consistent structure:

```json
{
  "success": true,
  "data": {
    // Response data
  },
  "creditsCost": 1
}
```

For detailed information about endpoints, error handling, and rate limits, see the [API overview](/api-reference/overview).
